# Магазин (Shop Application)

Графическое приложение на языке Python, предназначенное для имитации процесса покупки товаров в магазине.  
Проект разработан в рамках курсовой работы и ориентирован на демонстрацию принципов объектно-ориентированного программирования, проектирования архитектуры программного продукта и разработки графического пользовательского интерфейса.

---

## Описание проекта

Данное приложение моделирует поведение покупателя в магазине. Пользователь имеет возможность добавлять товары в корзину, взвешивать весовые товары, удалять товары из корзины, оплачивать покупку наличными средствами и бонусными баллами, а также получать чек после завершения операции покупки.

Программа разработана с использованием стандартных средств языка Python и не требует установки сторонних библиотек.

---

Программа предназначена для учебного использования и может быть применена в качестве примера реализации программного продукта среднего уровня сложности. Основное назначение проекта заключается в закреплении навыков объектно-ориентированного программирования, работы с графическим интерфейсом пользователя и реализации бизнес-логики.

---

## Функциональные возможности

Программа предоставляет пользователю следующие возможности:

- формирование корзины покупок;
- добавление штучных и весовых товаров;
- обязательное взвешивание весовых товаров перед покупкой;
- удаление выбранных товаров из корзины;
- автоматический расчёт стоимости товаров и общей суммы покупки;
- оплата покупки наличными средствами;
- оплата покупки бонусными баллами (полностью или частично);
- формирование чека после успешной оплаты;
- информирование пользователя об ошибках и некорректных действиях.

---

## Пользовательский интерфейс

Графический пользовательский интерфейс реализован с использованием библиотеки Tkinter и выполнен в светлой цветовой гамме с тёмно-зелёными акцентами.

Интерфейс главного окна включает следующие элементы:

- список товаров, добавленных в корзину;
- кнопки добавления товаров;
- поле ввода веса для весовых товаров;
- кнопку взвешивания выбранного товара;
- кнопку удаления товара из корзины;
- кнопки выбора способа оплаты;
- информационную строку с текущим балансом наличных средств и бонусных баллов.

Интерфейс ориентирован на интуитивное взаимодействие и не требует специальных навыков для работы с программой.

---

## Порядок работы с программой

После запуска приложения пользователю отображается главное окно магазина. Для начала работы необходимо добавить товары в корзину с помощью соответствующих кнопок. Добавленные товары отображаются в списке корзины.

Для весовых товаров требуется обязательное указание веса. Для этого пользователь должен выбрать соответствующий товар в корзине, ввести значение веса в килограммах и нажать кнопку взвешивания. После этого стоимость товара автоматически пересчитывается и отображается в корзине.

При необходимости пользователь может удалить любой товар из корзины, выбрав его в списке и нажав кнопку удаления.

После формирования корзины пользователь может перейти к оплате покупки, выбрав подходящий способ оплаты. Программа поддерживает оплату наличными средствами и бонусными баллами. В случае успешной оплаты корзина очищается, а пользователю отображается чек.

---

## Формирование чека

После завершения оплаты программа автоматически формирует чек в отдельном окне. Чек содержит следующую информацию:

- перечень приобретённых товаров;
- стоимость каждого товара;
- итоговую сумму покупки;
- сумму, оплаченную бонусными баллами;
- сумму, оплаченную наличными средствами.

Чек предназначен для визуального подтверждения корректности выполненной операции.

---

## Обработка ошибок

В программе реализована система обработки ошибок, которая предотвращает выполнение некорректных действий. Обрабатываются следующие ситуации:

- попытка оплаты покупки при наличии невзвешенного весового товара;
- ввод некорректного значения веса;
- попытка выполнения действий без выбора товара;
- недостаток средств для оплаты покупки.

Во всех перечисленных случаях пользователю выводится информативное сообщение с описанием причины ошибки.
