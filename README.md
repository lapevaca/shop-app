Описание проекта

Данное приложение представляет собой графическую программу, имитирующую процесс покупки товаров в магазине. Пользователь может добавлять товары в корзину, взвешивать весовые товары, оплачивать покупку наличными средствами и бонусными баллами, а также получать чек после завершения покупки.

Программа разработана на языке Python с использованием библиотеки Tkinter и предназначена для учебных целей, в частности для выполнения курсовой работы.

Системные требования

Python версии 3.9 и выше

Операционная система: Windows / Linux / macOS

Дополнительные библиотеки не требуются

Установка и запуск

Склонировать репозиторий:

git clone https://github.com/username/shop-app.git


Перейти в каталог проекта:

cd shop-app


Запустить программу:

python main.py

Основные возможности программы

добавление товаров в корзину;

поддержка штучных и весовых товаров;

взвешивание товаров с указанием веса в килограммах;

удаление товаров из корзины;

оплата покупки наличными средствами;

оплата покупки с использованием бонусных баллов (полностью или частично);

автоматический расчёт итоговой стоимости;

формирование чека в отдельном окне;

обработка ошибок пользовательского ввода.

Пользовательский интерфейс

Главное окно приложения состоит из следующих элементов:

область отображения корзины покупок;

кнопки добавления товаров;

поле ввода веса для весовых товаров;

кнопка взвешивания товара;

кнопка удаления товара из корзины;

кнопки оплаты покупки;

информационная строка с балансом наличных средств и бонусов.

Интерфейс выполнен в светлой цветовой гамме с тёмно-зелёными акцентами для повышения удобства восприятия.

Порядок работы с программой

После запуска приложения пользователю доступно главное окно магазина. Для формирования покупки необходимо добавить товары в корзину, используя соответствующие кнопки. Добавленные товары отображаются в списке корзины с указанием их стоимости или статуса взвешивания.

Для весовых товаров требуется обязательное указание веса. Для этого необходимо выбрать товар в корзине, ввести значение веса в килограммах и нажать кнопку взвешивания. После выполнения данной операции стоимость товара автоматически пересчитывается.

При необходимости пользователь может удалить любой товар из корзины, выбрав его в списке и нажав кнопку удаления.

После формирования корзины пользователь может перейти к оплате покупки. Программа поддерживает оплату наличными средствами, а также оплату с использованием бонусных баллов. В случае недостатка средств или попытки оплаты невзвешенного товара программа уведомит пользователя об ошибке.

После успешной оплаты формируется чек, отображаемый в отдельном окне, содержащий перечень товаров, их стоимость, итоговую сумму покупки и информацию о способе оплаты.

Обработка ошибок

В программе предусмотрена обработка следующих ситуаций:

попытка оплаты покупки без взвешивания весового товара;

ввод некорректного значения веса;

попытка оплаты при недостатке средств;

попытка взвешивания или удаления товара без его выбора.

Во всех перечисленных случаях пользователю выводится информативное сообщение с описанием ошибки.

Пример сценария использования

Запустить программу.

Добавить в корзину весовой товар (например, яблоки).

Выбрать добавленный товар и указать вес.

Добавить штучный товар (например, хлеб).

Проверить итоговую сумму.

Оплатить покупку с использованием бонусов.

Ознакомиться с сформированным чеком.
